# Modul Pembelajaran Golang (Go)

Selamat datang di Modul Pembelajaran Golang! Modul ini dirancang khusus untuk mulai belajar Go dari nol, dengan materi yang dipecah-pecah agar mudah dibaca dan dipahami di GitHub.

## Daftar Isi Materi

Berikut adalah urutan materi yang bisa kamu pelajari. Klik pada masing-masing link untuk mulai membaca materi:

1. **[Dasar-Dasar Golang](./01-Dasar-Dasar/README.md)**
   - Pengenalan singkat Golang.
   - Program pertama (Hello World).
   - Variabel, Konstanta, dan Tipe Data.

2. **[Array dan Slice](./02-Array-dan-Slice/README.md)**
   - Perbedaan Array dan Slice di Go.
   - Cara mendeklarasikan dan mengakses elemen.
   - Manipulasi data menggunakan Slice.

3. **[Algoritma Dasar (Sorting & Searching)](./03-Algoritma-Dasar/README.md)**
   - Menerapkan algoritma sederhana menggunakan Golang.
   - Contoh implementasi algoritma **Sort** (Bubble Sort, Selection Sort).

4. **[Struktur Kontrol (If, Switch, For)](./04-Struktur-Kontrol/README.md)**
   - Percabangan logika dengan `if-else` dan `switch`.
   - Melakukan perulangan menggunakan `for`.

5. **[Fungsi (Function)](./05-Fungsi/README.md)**
   - Cara memecah kode besar menjadi fungsi kecil.
   - Parameter dan nilai kembalian (return values).

6. **[Pointer](./06-Pointer/README.md)**
   - Memahami memori di Go.
   - Cara membaca reference (`&`) dan mengirim data lebih efisien.

7. **[Struct dan Method](./07-Struct-dan-Method/README.md)**
   - Mengelompokkan data yang kompleks (Mirip Class/OOP).
   - Menambahkan fungsi khusus di dalam data (`method`).

8. **[Goroutine dan Channel](./08-Goroutine-dan-Channel/README.md)**
   - Menjalankan proses secara paralel (antrian) dengan sangat ringan.
   - Kirim pesan antar proses menggunakan Channel.

9. **[Error Handling](./09-Error-Handling/README.md)**
   - Menangani pesan kegagalan secara terstruktur tanpa membuat program *crash*.

10. **[Go Modules](./10-Go-Modules/README.md)**
    - Membuat project berstandar resmi (go mod init).
    - Menarik modul (library) dari internet.

---
