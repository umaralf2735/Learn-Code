# Modul Pembelajaran JavaScript (JS)

Selamat datang di Modul JavaScript! JavaScript (JS) adalah bahasa pemrograman paling populer di dunia yang awalnya dibuat untuk membuat *website* menjadi hidup dan interaktif, namun sekarang JS bisa dipakai untuk membuat *backend* (Node.js), aplikasi mobile (React Native), hingga *game*.

5. **[Algoritma Dasar](./05-Algoritma-Dasar/README.md)**
