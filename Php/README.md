# Modul Pembelajaran PHP

Selamat datang di modul PHP! PHP (PHP: Hypertext Preprocessor) adalah rajanya website dan server internet era 2000-an dan hingga detik ini menguasai lebih dari 70% server website di dunia (Berkat WordPress dan Laravel). PHP sangat khusus didesain untuk satu tujuan: Mengolah halaman Web.

   - Sintaks pembuka gaib `<?php`, `echo`, dan deklarasi variabel tanda dollar (`$`).
2. **[Struktur Kontrol](./02-Struktur-Kontrol/README.md)**
   - Logika `if-else` dan perulangan `while`/`for`.
3. **[Array dan Assosiatif](./03-Array-Assosiatif/README.md)**
   - Kekuatan tempur PHP dalam menangani barisan data Dictionary.
4. **[Fungsi di PHP](./04-Fungsi/README.md)**
   - Membongkar penulisan *Function* kembalian.
5. **[Include dan Require](./05-Include-Require/README.md)**
   - Menyerap belahan file-file PHP terpisah ke satu halaman.
6. **[Pemrograman OOP Dasar](./06-OOP-Dsr/README.md)**
   - Mencetak objek dengan Class (Lahirnya Framework Modern).
7. **[Penanganan Form (POST & GET)](./07-Form-Method/README.md)**

8. **[Session dan Cookies](./08-Session-Cookie/README.md)**
