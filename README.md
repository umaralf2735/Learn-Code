## 🗺️ Peta Jalur Belajar (Learning Path) yang Direkomendasikan

### 🥇 Langkah 1: Pahami Mesin Intinya Dahulu (Bahasa C)
Jangan langsung gila-gilaan bikin website atau Game 3D. Kamu WAJIB hukumnya paham *bagaimana komputer sebenarnya berpikir mengolah RAM dan memori*. 
- 👉 **[Masuk ke Folder Bahasa C](./C)**
- **Kenapa?** C adalah bapak dari segala bahasa di bawah ini. Walau pusing karena galak, C akan membentuk mental logika mu sekeras baja. Kamu jadi paham array kaku mutlak dan perulangan murni.

### 🥈 Langkah 2: Pilih Cabang Karir Utamamu!
Setelah lulus dari siksaan C (setidaknya kamu paham variabel, Array, Loop, dan Fungsi C), saatnya kamu menunjuk **Keahlian Duniamu**. Pilih salah satu cabang (Track) di bawah.

#### 🌐 Track A: Fullstack Web (Mau Bikin Website Keren / Toko Online?)
Kalo hidupmu didedikasikan buat Frontend (tampilan UI) dan Backend Web:
1. Pelajari **[JavaScript](./Javascript)**: Jantung mutlak bagi website yang bisa diklik.
2. Lanjutkan ke **[PHP](./Php)**: Penguasa server website legendaris yang mem-backup database ke format HTML. (Lebih dari 70% web sejagat pakai ekosistem ini).
3. Setelah jago JS, pindahlah ke **[TypeScript](./Typescript)**: Ini adalah standar Industri Web Modern yang bikin gajimu melesat tajam.

#### 🤖 Track B: Data Science, AI & Hacking (Mau main Robot & Data?)
Kalo kamu benci desain tampilan, dan sukanya mbedah Algoritma Data Karyawan, Prediksi Cuaca, dan Mesin Kecerdasan Buatan (AI):
1. Masuk ke **[Python](./Python)**. Ini adalah dewa mutlaknya. Bahasanya semudah bahasa Inggris tanpa banyak titik koma. Disinilah ekosistem AI dunia berkumpul.
2. Sebagai alternatif selingan yang seru dan "nyeningin developer", kamu bisa iseng icip-icip **[Ruby](./Ruby)**.

#### 🎮 Track C: Enterprise Backend & AAA Games (Mau garap Aplikasi Desktop / Game 3D Bangsa Playstation?)
Ini kasta paling berat. Kalo lo suka arsitektur koding yang super ketat ala proyek ratusan millyar Dollar atau mau bikin game di Unreal Engine / Unity:
1. Sikat **[C++](./C++)**: Pelanjutnya C tapi dengan senjata sakti *OOP*. Wajib buat Game Developer AAA dan Robotics tingkat atas.
2. Belajar **[C# (C-Sharp)](./C%23)**: Dewanya Microsoft. Super enak dibaca dan merupakan bahasa murni dari Unity Engine (Game dev Mobile tersukses saat ini).
3. Serap ilmu **[Golang](./Golang)**: Bahasa racikan Google untuk performa *Backend Server* level dewa dengan kelincahan tingkat tinggi menangani antrian jutaan user sekaligus (Concurrency).
