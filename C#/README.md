# Modul Pembelajaran C# (C-Sharp)

Selamat datang di modul C#! Bahasa andalan utama buatan **Microsoft** yang menjadi tulang punggung game sedunia (Unity Engine), pembuatan aplikasi desktop Windows, hingga *Backend Website Enterprise* melalui kerangka .NET. 

Bahasa ini adalah titisan modern dari C++ dengan rasa dan keelokan penulisan semulus Java.

## Daftar Isi Materi

1. **[Dasar-Dasar C#](./01-Dasar-Dasar/README.md)**
   - Struktur murni `Program.cs`, kompilasi, dan deklarasi variabel lekat.
2. **[Struktur Kontrol C#](./02-Struktur-Kontrol/README.md)**
   - Pengecekan kondisi dan perulangan.
3. **[Array dan List Fleksibel](./03-Array-List/README.md)**
   - Koleksi penampung standar `[]` dan sang penggeser takhta `List<T>`.
4. **[Method di C#](./04-Method/README.md)**
   - Menghidupkan aksi terpisah pakai Function/Method.
5. **[Class OOP Sejati](./05-OOP-Class-Object/README.md)**
   - Mendesain cetakan Blueprint secara mutlak (C# adalah 100% bahasa OOP, nyaris ga bisa nulis tanpa Class).
6. **[Encapsulation & Properties](./06-Encapsulation-Property/README.md)**
   - Fitur kebanggaan C# `{ get; set; }` pengganti ngetik panjang getter/setter.
7. **[Pewarisan & Polimorfisme](./07-Inheritance-Polymorphism/README.md)**
   - Mengambil DNA induk class (`:`) dan melangkahi perintah Aslinya (`override`).
8. **[Interface (Kavling Kontrak) ](./08-Interface/README.md)**
   - Memaksa sebuah class mengikuti cetakan protokol wajib.
9. **[LINQ (Senjata Rahasia Maut)](./09-LINQ-Collection/README.md)**
   - Sihir merombak Array/List hanya dalam 1 baris tanpa Foreach (SQL style).
10. **[Error Handling (.NET)](./10-Error-Exception/README.md)**
    - Jurus `try...catch` menahan gempuran fatal layaknya pro.
