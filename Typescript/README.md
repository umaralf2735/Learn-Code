# Modul Pembelajaran TypeScript (TS)

Selamat datang di modul TypeScript! Singkatnya, Typescript adalah **"JavaScript yang Diperketat dengan Aturan"**. 

Diciptakan oleh Microsoft, bila Javascript murni (Vanilla) sangat pemalas dan bebas gonta-ganti tipe data (yang rentan melahirkan Error Bisu), maka Typescript memaksa kalian memahat tipe datanya dari awal layaknya Golang/C++. Inilah raja bahasa *Frontend Modern* abad ini.
