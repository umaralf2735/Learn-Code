# Modul Pembelajaran Ruby

Selamat datang di modul Ruby! Ruby adalah bahasa pemrograman yang didesain agar **"Programmer Merasa Bahagia"**. Penulisannya sangat mirip dengan bahasa Inggris manusia yang natural.

## Daftar Isi Materi

Berikut tingkat penguasaan Ruby yang bisa kau lahap:

1. **[Dasar-Dasar Ruby](./01-Dasar-Dasar/README.md)**
   - `puts`, variabel, *"Everything is an Object"*.
2. **[Struktur Kontrol (If, Unless, Loop)](./02-Struktur-Kontrol/README.md)**
   - Percabangan super elegan dan `.times`.
3. **[Array dan Hash](./03-Array-dan-Hash/README.md)**
   - List data, Hashes bersimbol dan `.each`.
4. **[Metode dan Blok](./04-Metode-dan-Blok/README.md)**
   - Fungsi dan rahasia si `yield` block.
5. **[Algoritma Dasar (Enumerable Magic)](./05-Algoritma-Dasar/README.md)**
   - `.select`, `.map`, `.find`.
6. **[Pemrograman OOP (Class)](./06-OOP-Class/README.md)**
   - Mendesain class dengan *Instance Variable* (`@`).
7. **[Module dan Mixin](./07-Modul-dan-Mixin/README.md)**
   - Menginjeksi *Namespace* kedalam Class.
8. **[Error Handling](./08-Error-Handling/README.md)**
   - Begin/Rescue/Ensure agar websitemu tak kolaps.
9. **[File I/O (Bongkar File)](./09-File-IO/README.md)**
   - Mengubah text file murni tanpa merusak *formatting*.
10. **[Ruby Gems (Ekosistem Library)](./10-Ruby-Gems/README.md)**
    - Berkenalan dengan toko sihir sakti komunitas, RubyGems dan Bundler!
