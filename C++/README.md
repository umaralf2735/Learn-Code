# Modul Pembelajaran C++

Selamat datang di ranah dewa bahasa C++! 
Ini adalah versi Upgrade dari pendahulunya sang bahasa C. Jika C cuma sekedar terstruktur procedural, maka C++ hadir mengawinkan kecepatan roket memori milik C dengan arsitektur sakti Pemrograman Objek Modern (OOP).

Bahasa ini adalah roh mutlak pembuatan Sistem Operasi Desktop, Game AAA (Unreal Engine), dan Software super berat layaknya Adobe Premiere & Photoshop.

## Daftar Isi Materi

1. **[Dasar-Dasar C++](./01-Dasar-Dasar/README.md)**
   - Header ajaib `iostream`, perintah cetak modern `cout` & `cin`.
2. **[Struktur Kontrol](./02-Struktur-Kontrol/README.md)**
   - Pengecekan kondisi dan perulangan standar.
3. **[Array vs Vectors](./03-Array-Vector/README.md)**
   - Move on dari Array Kuno ke pelindung sihir memori, `std::vector`.
4. **[Fungsi dan Reference](./04-Fungsi-Reference/README.md)**
   - Trik curang memanipulasi parameter asli pakai `&` (Pass by Reference) yang lebih ramah dari Pointer.
5. **[Pointers dan Memori Buatan](./05-Pointer-Memori/README.md)**
   - Hal yang bikin orang mundur: Mengendalikan alamat RAM mentah `*` dan `new/delete`.
6. **[Class OOP Mutlak](./06-OOP-Class-Dasar/README.md)**
   - C++ adalah OOP level dewa. Inilah pondasinya.
7. **[Inheritance & Polymorphism](./07-Inheritance-Polymorphism/README.md)**
   - Menurunkan takhta class ke ras yang berbeda (Fitur Game Component).
8. **[Standard Template Library/STL](./08-STL-Algorithm/README.md)**
   - Karena ngetik manual Algoritma melelahkan, STL buatan C++ menyediakan Magic Sort & Find 1 baris.
9. **[Exception Handling](./09-Error-Exception/README.md)**
   - Mengurung resiko meledaknya Blue Screen pakai Try..Catch (Hal yang tidak ada di bapanya C).
10. **[Arsitektur Proyek (Header `.h`)](./10-Header-File/README.md)**
    - Berhenti nulis 10,000 baris di `main.cpp`. Pisahkan judul cetakan dan cara kerjanya secara rahasia!
