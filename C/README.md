# Modul Pembelajaran Bahasa C

Selamat datang di modul Bahasa C! Jika kamu mengibaratkan bahasa pemrograman adalah mobil, maka bahasa C adalah "Mesinnya Mobil". Sebagian besar bahasa modern populer seperti C++, Java, bahkan compiler dari Golang dan Ruby itu dulunya dibuat menggunakan pondasi bahasa C.

Belajar C akan membuatmu sangat menguasai bagaimana komputer bekerja dari segi manajemen memori secara nyata.

6. **[Algoritma Dasar](./06-Algoritma-Dasar/README.md)**
